# Упражнения на работу со списками в Python

Эти задания помогут вам освоить базовые операции со списками: использование индексов, срезов, а также добавление и удаление элементов.

## Цель

Вам предстоит реализовать несколько функций в файле `main.py`. Каждая функция — это небольшая задача на отработку конкретного навыка.

**Важно:** В этом задании мы начинаем работать с функциями. Обратите внимание на комментарии в `main.py` — они объясняют, как брать данные для работы и как возвращать результат.

## Инструкция

### Шаг 1: Настройте окружение

```bash
python -m venv venv
source venv/Scripts/activate
pip install -r requirements.txt
```

### Шаг 2: Выполните задания в `main.py`

Откройте `main.py` и реализуйте следующие функции:

1.  **`get_first_and_last_element(lst)`**: Должна вернуть список, состоящий из первого и последнего элемента.

2.  **`get_middle_elements(lst)`**: Должна вернуть список со всеми элементами, кроме первого и последнего.

3.  **`get_elements_from_second_to_fourth(lst)`**: Должна вернуть элементы со второго по четвертый (включительно).

4.  **`add_element_to_start_and_end(lst, element)`**: Должна добавить один и тот же элемент в начало и в конец списка.

5.  **`remove_second_element(lst)`**: Должна удалить второй элемент из списка.

### Шаг 3: Запустите тесты

После реализации функций запустите тесты, чтобы проверить себя:

```bash
pytest
```

Тесты покажут, какие задания выполнены верно, а где есть ошибки.
